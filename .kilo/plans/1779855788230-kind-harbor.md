# Plan: Summary Dashboard — 新增 Brand / Product Desc 欄位

## 目標
在「統計摘要 (Summary Dashboard)」的「按Article統計」表中加入 **Brand** 和 **Product Desc** 兩個欄位，令用戶更易閱讀。

**現有欄位:** 按Article統計 | 總調貨件數 | 調貨行數 | 涉及OM數量  
**目標欄位:** Brand | 按Article統計 | Product Desc | 總調貨件數 | 調貨行數 | 涉及OM數量

---

## 需要修改的檔案 (3 個)

### 1. `services/statistics.py` — 資料聚合層 (line 16–25)

**現狀:** `article_stats` 只記錄 `total_qty`, `count`, `oms`，沒有 Brand/Product Desc。

**修改:** 在建立 `article_stats` dict 時，同時從 `rec` 取出 `Brand` 和 `Product Desc`：

```python
article_stats = {}
for rec in recommendations:
    article = rec['Article']
    if article not in article_stats:
        article_stats[article] = {
            'brand': rec.get('Brand') or rec.get('Product Hierarchy', ''),
            'product_desc': rec.get('Product Desc', ''),
            'total_qty': 0,
            'count': 0,
            'oms': set(),
        }
    article_stats[article]['total_qty'] += rec['Transfer Qty']
    article_stats[article]['count'] += 1
    article_stats[article]['oms'].add(rec['Transfer OM'])
```

### 2. `ui/display.py` — Streamlit 介面 (lines 237–245)

**現狀:** DataFrame 欄位 `Article | Total Qty | Count | OM Count`。

**修改:** 新增 `Brand` 和 `Product Desc` 欄：

```python
article_df = pd.DataFrame([
    {
        'Brand': stats.get('brand', ''),
        'Article': article,
        'Product Desc': stats.get('product_desc', ''),
        'Total Qty': stats['total_qty'],
        'Count': stats['count'],
        'OM Count': stats['om_count'],
    }
    for article, stats in article_stats.items()
])
```

### 3. `excel_generator.py` — Excel 輸出 (lines 249–262)

**現狀:** 表頭 A–D，資料從 A 開始寫。

**修改:** 插入 Brand (col A) 和 Product Desc (col C)，將原有欄位右移：

| 欄 | 內容 |
|---|---|
| A | Brand |
| B | 按Article統計 (Article) |
| C | Product Desc |
| D | 總調貨件數 |
| E | 調貨行數 |
| F | 涉及OM數量 |

```python
row = 8
worksheet.write(f'A{row}', 'Brand', header_format)
worksheet.write(f'B{row}', '按Article統計', header_format)
worksheet.write(f'C{row}', 'Product Desc', header_format)
worksheet.write(f'D{row}', '總調貨件數', header_format)
worksheet.write(f'E{row}', '調貨行數', header_format)
worksheet.write(f'F{row}', '涉及OM數量', header_format)

article_stats = statistics.get('article_stats', {})
for article, stats in article_stats.items():
    row += 1
    worksheet.write(f'A{row}', stats.get('brand', ''), data_format)
    worksheet.write(f'B{row}', article, data_format)
    worksheet.write(f'C{row}', stats.get('product_desc', ''), data_format)
    worksheet.write(f'D{row}', stats['total_qty'], data_format)
    worksheet.write(f'E{row}', stats['count'], data_format)
    worksheet.write(f'F{row}', stats['om_count'], data_format)
```
