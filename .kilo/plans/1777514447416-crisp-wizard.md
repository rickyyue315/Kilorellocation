# 新增「精簡SKU」模式計劃

## 概述
新增兩個精簡SKU模式變體（系統從22模式擴展為24模式）：
| 模式代碼 | 模式名稱 | 描述 |
|----------|----------|------|
| `精簡SKU(限同OM)` | 精簡SKU(限同OM) | 限制同OM轉貨，剩餘退回D001 |
| `精簡SKU(跨OM)` | 精簡SKU(跨OM) | 跨OM轉貨（Windy限同OM），剩餘退回D001 |

## 核心業務規則

### 1. RF店舖存貨上限
```python
Cap = max(Safety Stock × 2, Last 2 Month Sold Qty × 2)
```

### 2. 轉出來源 (identify_sources)

| 優先級 | 來源類型 | 條件 | 可轉數量 | Source Type |
|--------|----------|------|----------|-------------|
| 1 | ND店舖 | SaSa Net Stock > 0 | 全數庫存 | `精簡SKU ND轉出` |
| 2 | RF店舖 | (SaSa Net Stock + Pending Received) > Cap 且非最高動銷店 | min(total_available - Cap, SaSa Net Stock) | `精簡SKU RF轉出` |

- 保護最高動銷RF店舖不被轉出（同現有模式邏輯）
- 若所有RF店舖銷量相同，則不保護任何店舖
- 按priority排序：ND優先，RF次之

### 3. 接收目標 (identify_destinations)

| 條件 | 需求數量 | 接收上限 | Dest Type |
|------|----------|----------|-----------|
| 僅RF店舖 | | | |
| (SaSa Net Stock + Pending Received) < Cap | Cap - total_available | Cap - total_available | `精簡SKU接收` |

- 僅RF可接收（ND不可接收）
- 無特殊優先級排序

### 4. 轉貨量限制（參考C1模式）

- 轉給RF店舖時：
  - 來源 `transferable_qty >= 2` 才參與配對（參照C1 `actual_transferable < 2` 跳過邏輯）
  - 來源按可轉量降序排序，優先使用可轉量較大的來源（減少拆單）
- 退回D001：無最低數量限制

### 5. D001退回機制（第二階段）

完成所有RF-to-RF配對後，遍歷所有來源：
- 若來源仍有 `transferable_qty > 0`，生成一筆退回D001記錄：
  ```python
  {
      'Transfer Site': source['site'],
      'Transfer OM': source['om'],
      'Receive Site': 'D001',
      'Receive OM': source['om'],          # 沿用source OM
      'Transfer Qty': source['transferable_qty'],  # 剩餘全數
      'Source Type': source['source_type'],
      'Destination Type': '退回D001',
      'Notes': '精簡SKU模式：剩餘庫存退回D001'
  }
  ```

### 6. 同OM vs 跨OM 差異

| 特性 | 精簡SKU(限同OM) | 精簡SKU(跨OM) |
|------|-----------------|---------------|
| 分組方式 | `groupby(['Article', 'OM'])` | `groupby(['Article'])` |
| 同OM限制 | 僅同OM配對 | 不限制跨OM |
| Windy限制 | N/A | Windy轉出只能到Windy接收 |
| HD限制 | N/A | HD不能轉到HA/HB/HC |
| 店數限制 | 不適用 | 不適用 |
| D001退回 | ✅ | ✅ |

def _calc_sku_cap(self, row_or_data) -> int:
    safety = int(row_or_data['Safety Stock'] if isinstance(row_or_data, dict) and 'Safety Stock' in row_or_data else row_or_data.get('Safety Stock', 0) or 0) if isinstance(row_or_data, dict) else int(row_or_data['Safety Stock'])
    last_2_month = int(row_or_data.get('last_2_month_sold_qty', 0) or 0) if isinstance(row_or_data, dict) else int(row_or_data['Last 2 Month Sold Qty'] if 'Last 2 Month Sold Qty' in row_or_data.index else 0)
    return max(safety * 2, last_2_month * 2)

### Step 4: `business_logic.py` — 新增 `_match_transfers_simplified_sku` 方法

在 `_match_transfers_c2_mode` 方法附近（約第1281行後）新增：

```python
def _match_transfers_simplified_sku(self, sources, destinations, article, product_desc, mode, om=None):
    """
    精簡SKU模式匹配邏輯：
    Phase 1: 常規RF-to-RF配對（參照C1：最少2件起轉，優先可轉量較大來源）
    Phase 2: 剩餘退回D001（無數量限制）
    
    - 限同OM：僅同OM配對
    - 跨OM：允許跨OM，Windy限同OM，HD不能轉HA/HB/HC
    """
    recommendations = []
    cross_om = (mode == self.mode_simplified_sku_cross)
    
    temp_sources = [s.copy() for s in sources]
    for s in temp_sources:
        s['total_transferred'] = 0
    temp_destinations = [d.copy() for d in destinations]
    
    transfer_sites = set()
    receive_sites = set()
    received_qty_by_site = {}
    
    # Phase 1: 常規配對
    # C1模式：按可轉量降序，優先使用可轉量較大的來源（減少拆單）
    pending_sources = sorted(
        [s for s in temp_sources if s['transferable_qty'] >= 2],
        key=lambda x: -x['transferable_qty']
    )
    
    for source in pending_sources:
        for dest in temp_destinations:
            if source['transferable_qty'] <= 0 or dest['needed_qty'] <= 0:
                continue
            if source['site'] == dest['site']:
                continue
            if dest['site'] in transfer_sites:
                continue
            if source['site'] in receive_sites:
                continue
            if dest.get('rp_type') == 'ND':
                continue
            
            # 限同OM：檢查OM是否相同
            if not cross_om and source['om'] != dest['om']:
                continue
            
            # 跨OM：Windy只能轉到Windy
            if cross_om and source.get('om') == 'Windy' and dest.get('om') != 'Windy':
                continue
            
            # 跨OM：HD不能轉到HA/HB/HC
            if cross_om:
                source_site = source['site']
                is_source_hd = isinstance(source_site, str) and source_site.upper().startswith('HD')
                if is_source_hd:
                    dest_site_upper = dest['site'].upper() if isinstance(dest['site'], str) else ''
                    if dest_site_upper.startswith(('HA', 'HB', 'HC')):
                        continue
            
            # 檢查接收上限
            receive_site_key = f"{dest['site']}_{article}"
            current_received = received_qty_by_site.get(receive_site_key, 0)
            max_receive = dest.get('max_receive_qty', float('inf'))
            if current_received >= max_receive:
                continue
            
            transfer_qty = min(source['transferable_qty'], dest['needed_qty'], max_receive - current_received)
            if transfer_qty <= 0:
                continue
            
            # 建立建議
            rec = {
                'Article': article, 'Product Desc': product_desc,
                'Transfer OM': source['om'], 'Transfer Site': source['site'],
                'Receive OM': dest['om'], 'Receive Site': dest['site'],
                'Transfer Qty': transfer_qty,
                'Original Stock': source['original_stock'],
                'After Transfer Stock': source['original_stock'] - source.get('total_transferred', 0) - transfer_qty,
                'Safety Stock': 0, 'MOQ': 0,
                'Source Priority': source['priority'], 'Destination Priority': dest['priority'],
                'Source Type': source['source_type'], 'Destination Type': dest['dest_type'],
                'Notes': self._create_recommendation_note(source, dest, current_received, transfer_qty, mode),
                'Transfer Site Last Month Sold Qty': source.get('last_month_sold_qty', 0),
                'Transfer Site MTD Sold Qty': source.get('mtd_sold_qty', 0),
                'Receive Site Last Month Sold Qty': dest.get('last_month_sold_qty', 0),
                'Receive Site MTD Sold Qty': dest.get('mtd_sold_qty', 0),
                'Receive Original Stock': dest.get('current_stock', 0)
            }
            if 'target_qty' in dest:
                rec['Target Qty'] = dest['target_qty']
            rec['Cumulative Received Qty'] = current_received + transfer_qty
            
            recommendations.append(rec)
            
            received_qty_by_site[receive_site_key] = current_received + transfer_qty
            source['total_transferred'] = source.get('total_transferred', 0) + transfer_qty
            source['transferable_qty'] -= transfer_qty
            dest['needed_qty'] -= transfer_qty
            
            if dest['needed_qty'] <= 0:
                dest['needed_qty'] = 0
    
    # Phase 2: 退回D001（所有剩餘未轉出數量）
    for source in temp_sources:
        remaining = source['transferable_qty']
        if remaining <= 0:
            continue
        
        rec = {
            'Article': article, 'Product Desc': product_desc,
            'Transfer OM': source['om'], 'Transfer Site': source['site'],
            'Receive OM': source['om'], 'Receive Site': 'D001',
            'Transfer Qty': remaining,
            'Original Stock': source['original_stock'],
            'After Transfer Stock': source['original_stock'] - source.get('total_transferred', 0) - remaining,
            'Safety Stock': 0, 'MOQ': 0,
            'Source Priority': source['priority'], 'Destination Priority': 99,
            'Source Type': source['source_type'], 'Destination Type': '退回D001',
            'Notes': f"精簡SKU模式：剩餘庫存{remaining}件退回D001",
            'Transfer Site Last Month Sold Qty': source.get('last_month_sold_qty', 0),
            'Transfer Site MTD Sold Qty': source.get('mtd_sold_qty', 0),
            'Receive Site Last Month Sold Qty': 0,
            'Receive Site MTD Sold Qty': 0,
            'Receive Original Stock': 0,
            'Target Qty': 0,
            'Cumulative Received Qty': remaining
        }
        recommendations.append(rec)
    
    return recommendations
```
