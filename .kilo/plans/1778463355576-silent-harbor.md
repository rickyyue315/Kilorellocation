# Plan: F2 模式新增 HD 轉出選項

## 需求摘要

F2 模式目前禁止 HD 店舖轉貨到 HA/HB/HC。新增一個 F2 模式下的選項，讓使用者可以選擇「HD 可轉出（最後優先）」或維持「HD 不能轉出」（預設）。當選擇 HD 可轉出時，HD 轉出排在最低優先級，只有在其他來源都不夠時才會用到 HD。

---

## 修改清單

### 1. `business_logic.py` — 核心邏輯

#### 1a. 建構子新增參數
- `__init__` 新增 `f2_allow_hd_transfer: bool = False` 參數，存為 `self.f2_allow_hd_transfer`

#### 1b. `_match_transfers_f_mode` 修改 HD 限制邏輯
- **Line 1767-1769**：目前的 HD 限制是直接 `continue` 跳過
- 修改為：當 `mode == self.mode_f_target_only and self.f2_allow_hd_transfer` 時，**不跳過**但將 HD→HA/HB/HC 的配對排在最低優先級
- 實作方式：修改 `_f_source_sort_key` 函式，新增 tier 3 給 HD 轉到 HA/HB/HC 的情況
  - 現有 tier：0=ND, 1=同OM RF, 2=跨OM RF
  - 新增 tier：3=HD→HA/HB/HC（最低優先）
- 在 HD 限制檢查處改為：
  - 若 `self.f2_allow_hd_transfer and mode == self.mode_f_target_only`：不跳過，但在 sort key 中加入 HD 低優先標記
  - 否則：維持原來的 `continue` 行為

**具體實作：**
```python
# 在 _f_source_sort_key 中增加 hd_allow 參數
def _f_source_sort_key(src, dest_om, hd_allow=False):
    rp = str(src.get('rp_type', '')).upper()
    same_om = 1 if src.get('om') == dest_om else 2
    if rp == 'ND':
        tier = 0
    elif same_om == 1:
        tier = 1
    else:
        tier = 2
    # HD 轉到 HA/HB/HC 時，額外降級到 tier 4（最低優先）
    hd_penalty = 0
    if hd_allow and str(src.get('site', '')).upper().startswith('HD'):
        dest_site = ''  # 需要傳入
        # 這裡需要在排序時傳入 dest site
    return (tier, src.get('effective_sold_qty', 0))
```

**更好的實作方案：** 不修改 sort key，而是在 HD 檢查處做條件判斷：
- 將 sorted_sources 改為兩輪遍歷：第一輪跳過 HD→HA/HB/HC，第二輪才允許 HD→HA/HB/HC
- 或者更簡潔：在 sort key 中加入 HD 懲罰項

**最終方案：** 修改 sort key 加入 HD 懲罰，然後在 HD 限制檢查中，當 F2+allow 時不跳過：

```python
# Line 1715-1724 修改
def _f_source_sort_key(src, dest_om, dest_site=''):
    rp = str(src.get('rp_type', '')).upper()
    same_om = 1 if src.get('om') == dest_om else 2
    if rp == 'ND':
        tier = 0
    elif same_om == 1:
        tier = 1
    else:
        tier = 2
    hd_penalty = 0
    if str(src.get('site', '')).upper().startswith('HD') and dest_site.upper().startswith(('HA', 'HB', 'HC')):
        hd_penalty = 1  # 降級
    return (tier + hd_penalty, src.get('effective_sold_qty', 0))
```

但 sort key 裡無法直接知道 dest_site（在 for source in sorted_sources 循環外），需要改為在 sort lambda 中捕獲 dest：

```python
# Line 1741-1744 修改
sorted_sources = sorted(
    temp_sources,
    key=lambda s: _f_source_sort_key(s, dest.get('om', ''), dest.get('site', ''))
)
```

然後 HD 限制檢查處：
```python
# Line 1767-1769 修改
if self._is_hd_to_hk_restricted(source['site'], dest['site']):
    if not (mode == self.mode_f_target_only and self.f2_allow_hd_transfer):
        continue
```
